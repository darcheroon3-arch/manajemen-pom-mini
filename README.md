# Manajemen Pom Mini

Aplikasi Manajemen Pom Mini Profesional - Aplikasi kasir modern untuk manajemen pom bensin mini dengan fitur lengkap.

## Fitur

- **Dashboard** - Kartu statistik real-time (Sisa Bensin, Omzet, Profit, Pengeluaran, Balance)
- **Stok Bensin** - Tambah, edit, hapus stok dengan pengeluaran otomatis
- **Penjualan** - Input penjualan dengan auto-hitung omzet & profit
- **Pengeluaran** - 7 kategori pengeluaran dengan edit/hapus
- **Riwayat** - Audit log aktivitas lengkap dengan pencarian
- **Laporan** - Harian, Bulanan, Tahunan
- **Grafik** - Visualisasi data penjualan, omzet, profit, pengeluaran, balance
- **Pengaturan** - Harga jual, profit per liter, biaya isi, export CSV

## Tech Stack

- React Native (Expo)
- TypeScript
- Supabase (PostgreSQL)
- Lucide React Native Icons

## Data Storage

- **Kode sumber**: Tersimpan di GitHub
- **Data transaksi**: Tersimpan di Supabase Database (cloud, real-time)
- **Backup**: Export CSV tersedia di menu Pengaturan
